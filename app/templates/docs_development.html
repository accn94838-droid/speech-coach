Руководство по разработке

Краткие инструкции для разработчиков: настройка окружения, тестирование и стиль кода.

Установка и окружение

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

Тесты

Запустить тесты: pytest. Для покрытия: pytest --cov=app --cov-report=html.

Код-стиль

Используйте black, flake8, mypy для форматирования и проверки.
