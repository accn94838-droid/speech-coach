Развертывание

Инструкции по развертыванию Speech Coach на различные платформы.

Docker

Используйте Docker для изолированной среды: 

docker build -t speech-coach .
docker run -p 8000:8000 speech-coach

Облачные платформы

Поддерживаются все основные облачные платформы: AWS, Google Cloud, Azure, Heroku, DigitalOcean. Подробные инструкции см. в главной документации.
