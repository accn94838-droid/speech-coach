Быстрый старт

Следуйте шагам из Quickstart или используйте команды ниже на машине разработчика:

pip install -r requirements.txt
uvicorn app.main:app --reload

Содержание

  • Разработка — тесты, lint, требования
  • Безопасность и приватность данных

Где найти полную документацию

В репозитории присутствует развернутый набор документов: QUICKSTART, PROJECT_STRUCTURE, DEVELOPMENT и др. Для удобства сайта мы оформили краткую страницу — для полного чтения откройте папку docs/ в репозитории.

Если нужно, могу перенести отдельные разделы из `docs/` прямо в этот сайт и оформить развернутую навигацию.
