Полная установка

Детальное пошаговое руководство по установке Speech Coach на вашу систему.

Требования

  • Python 3.8+ (рекомендуется 3.10+)
  • pip (менеджер пакетов Python)
  • Git (для клонирования репозитория)
  • 4 GB RAM (минимум для работы моделей)
  • FFmpeg (для обработки видео/аудио)

Шаг 1: Клонируйте репозиторий

git clone https://github.com/nacc21332-netizen/speech-coach.git cd speech-coach

Шаг 2: Установите зависимости

Linux/macOS

python3 -m venv venv source venv/bin/activate pip install -r requirements.txt

Windows

python -m venv venv venv\Scripts\activate pip install -r requirements.txt

Шаг 3: Установите FFmpeg

Ubuntu/Debian

sudo apt-get install ffmpeg

macOS (с Homebrew)

brew install ffmpeg

Windows

Скачайте с ffmpeg.org и добавьте в PATH.

Шаг 4: Переменные окружения

Создайте файл .env в корневой директории:

LOG_LEVEL=INFO ALLOWED_VIDEO_EXTENSIONS=.mp4,.mov,.avi,.mkv,.flv,.mp3,.wav,.aac,.ogg MAX_FILE_SIZE_MB=500 GIGACHAT_API_KEY=your_api_key_here

Шаг 5: Запустите приложение

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

Приложение запустится на http://localhost:8000

Проверка установки

Проверьте здоровье приложения:

curl http://localhost:8000/health
