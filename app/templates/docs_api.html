REST API

Полная документация API Speech Coach для интеграции в ваши приложения.

Основные эндпоинты

Загрузка и анализ файла

POST /api/analysis/upload

Загрузить аудио или видео файл для анализа. Поддерживаются форматы MP4, MOV, MP3, WAV и другие.

Получение результатов анализа

GET /api/analysis/results/{analysis_id}

Получить результаты анализа по ID. Возвращает все метрики, временные отметки и рекомендации.

AI-коуч чат

POST /chat

Отправить сообщение в AI-коуч. Получить персональные рекомендации на основе анализа.

Примеры использования

Загрузка файла (Python)

import requests files = {'file': open('speech.mp4', 'rb')} response = requests.post('http://localhost:8000/api/analysis/upload', files=files) analysis_id = response.json()['analysis_id'] print(f"Analysis ID: {analysis_id}")

Получение результатов (JavaScript)

const analysisId = 'abc123'; fetch(`/api/analysis/results/${analysisId}`) .then(r => r.json()) .then(data => console.log(data.metrics));

Аутентификация

В текущей версии API доступен без авторизации. Для использования в production рекомендуется добавить API ключи. Подробнее см. в разделе Безопасность.

Обработка ошибок

API возвращает стандартные HTTP коды ошибок:

  • 400 — Bad Request (неверные параметры)
  • 404 — Not Found (файл не найден)
  • 413 — Payload Too Large (файл слишком большой)
  • 500 — Internal Server Error (ошибка сервера)
