Quick Start

Краткая инструкция по установке и запуску сервиса.

Требования

  • Python 3.8+
  • FFmpeg
  • Git

Установка

git clone 
cd speech-coach
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
mkdir -p logs cache
uvicorn app.main:app --reload

Запуск

После запуска сервис доступен на http://localhost:8000. OpenAPI: /docs.
