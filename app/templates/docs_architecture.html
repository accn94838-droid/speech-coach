Техническая архитектура

Описание компонентов и взаимодействия различных частей Speech Coach.

Основные компоненты

  • API сервер: FastAPI на Python
  • Распознавание речи: Whisper AI или другие модели
  • Анализ: Кастомные алгоритмы для метрик
  • AI-коуч: GigaChat для рекомендаций
  • Хранилище: PostgreSQL для данных

Поток данных

  1. Пользователь загружает видео/аудио
  2. Система преобразует в аудио (если нужно)
  3. Whisper распознает речь
  4. Анализаторы обрабатывают текст и аудио
  5. Результаты сохраняются в БД
  6. GigaChat генерирует рекомендации
  7. Пользователь видит результаты
