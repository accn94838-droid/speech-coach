Разработка

Руководство для разработчиков

Окружение

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

Тесты

Запустите тесты:

pytest

С покрытием:

pytest --cov=app --cov-report=html

Код-стиль

Используйте инструменты для проверки качества:

black app/
flake8 app/
mypy app/

Стандарты

  • Следуйте PEP 8
  • Добавляйте type hints на функции
  • Пишите документирующие строки (docstrings)
  • Обрабатывайте ошибки правильно
