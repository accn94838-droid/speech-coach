Структура проекта

Архитектура и организация кода

Основные директории

/app
  /api         - маршруты и эндпоинты
  /core        - конфигурация и утилиты
  /models      - Pydantic модели
  /services    - бизнес-логика (анализаторы, сервисы)
  /templates   - HTML шаблоны
  /static      - CSS, JS, статические файлы
  main.py      - точка входа FastAPI приложения

Ключевые компоненты

  • services/analyzer.py - основной анализатор речи
  • services/transcriber.py - распознавание речи (Whisper)
  • services/pipeline.py - оркестрация анализа
  • services/gigachat.py - интеграция с AI
  • services/cache.py - кеширование результатов

API маршруты

  • POST /api/v1/analyze - анализ файла
  • POST /chat - чат с AI
  • GET /health - проверка здоровья
